# React Native Core Skills

## Overview
This directory contains modular skill definitions for React Native development. Each skill represents a specific competency area with implementation patterns, best practices, and code examples.

## Available Skills

### 1. `expo-configuration.md`
Expo SDK setup, config plugins, custom dev clients, and build configuration.

### 2. `typescript-patterns.md`
Type-safe component patterns, navigation typing, API response typing, and generic hooks.

### 3. `native-navigation.md`
React Navigation v6+ stack, tabs, drawers, deep linking, and authentication flows.

### 4. `state-zustand.md`
Zustand store creation, persistence, middleware, and integration with React Query.

### 5. `api-client.md`
Axios setup, interceptors, retry logic, request cancellation, and type-safe API layers.

### 6. `offline-storage.md`
AsyncStorage, MMKV, WatermelonDB setup, data synchronization, and conflict resolution.

### 7. `styling-nativewind.md`
NativeWind configuration, responsive design, dark mode, and theme systems.

### 8. `stitch-design/SKILL.md` ⭐ NEW
Analyze Stitch mobile app designs and synthesize a semantic design system into DESIGN.md files for React Native development. Includes mobile-specific patterns, touch targets, safe areas, and platform adaptations.

### 9. `stitch-loop/SKILL.md` ⭐ NEW
Iteratively build React Native mobile app screens using Stitch with an autonomous baton-passing loop pattern. Integrates generated screens into Expo Router projects with type-safe navigation.

### 10. `form-handling.md`
React Hook Form integration, validation schemas (Zod/Yup), and error handling.

### 11. `image-optimization.md`
Image caching, lazy loading, optimization, and expo-image implementation.

### 12. `animation-reanimated.md`
React Native Reanimated 3, gesture handler, and complex animation patterns.

### 13. `push-notifications.md`
Expo Notifications, FCM/APNs setup, notification handlers, and deep link routing.

### 14. `biometric-auth.md`
Expo LocalAuthentication, biometric prompts, and secure credential storage.

### 15. `camera-media.md`
Expo Camera, image picker, video recording, and media library integration.

### 16. `location-services.md`
Expo Location, background tracking, geofencing, and map integration.

### 17. `testing-components.md`
Jest setup, Testing Library patterns, mock providers, and snapshot testing.

### 18. `e2e-detox.md`
Detox configuration, test writing, CI integration, and flaky test prevention.

### 19. `performance-profiling.md`
Flipper integration, Hermes profiling, bundle analysis, and memory optimization.

### 20. `security-hardening.md`
Secure storage, certificate pinning, obfuscation, and runtime protection.

### 21. `app-store-deployment.md`
App Store Connect, Google Play Console, versioning, and release management.

### 22. `ci-cd-pipeline.md`
EAS Build, GitHub Actions, automated testing, and deployment workflows.

## Usage
Reference these skills in your phase documentation or use them as standalone guides for implementing specific features.
